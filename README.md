# typefm

Monorepo for [type.fm](https://type.fm)

## Packages

- [@typefm/comrak-wasm](packages/comrak-wasm) — WebAssembly bindings for comrak markdown parser

## Setup

```bash
pnpm install
```

## Development

```bash
# Build all packages
pnpm -r build

# Run all tests
pnpm -r test
cargo test
```
